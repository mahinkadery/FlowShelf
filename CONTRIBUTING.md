# Contributing to FlowShelf

Thanks for your interest! FlowShelf is source-available (see [LICENSE.md](LICENSE.md)).
You're welcome to report bugs, suggest features, and submit code.

## Reporting bugs / ideas
Open a GitHub Issue with steps to reproduce, your macOS version, and screenshots
if relevant.

## Submitting code
1. Fork the repository.
2. Create a branch: `git checkout -b fix/short-description`.
3. Build & test locally:
   ```sh
   make install        # builds, signs, installs to /Applications
   ```
4. Keep changes focused; match the existing style and comment density.
5. Open a Pull Request describing what and why.
